在英文 Linux 中使用 Firefox 的印象笔记-悦读插件

Mar 8th, 2014

在知乎上看到这个问题，由于在英文 linux 的 firefox 中的 clearly 只能连接 evernote 账户而不能切换到中文的“印象笔记”，因此通过修改该插件的方式得到连接 “印象笔记” 的功能。

系统环境

系统: Linux arch-laptop 3.12.0-1-ARCH #1 SMP PREEMPT Wed Nov 6 09:06:27 CET 2013 x86_64 GNU/Linux Firefox 版本: 25 Clearly 版本: 9.4.7

修改步骤

1、安装 clearly， 如果之前用 clearly 登录过 evernote 账户， 需要在 Add-ons - clearly 的 preferences 里面 sing out， 或者卸载重装 clearly.

2、找到 clearly 插件的安装路径， 在 ~/.mozilla/firefox/sfhpk2t0.default/extensions 中， 其中 .mozilla/firefox/sfhpk2t0.default/extensions 可能不同.

3、该目录中有个名为 readable@evernote.com.xpi 的文件，这就 clearly 插件, 使用zip压缩.

4、用 unzip 解压缩该文件（建议复制该文件到另外的临时目录），命令 ：

5、使用文本编辑器修改如下文件：

5.1、修改 chrome/content/options/back.js 第 458 行， 修改 'has_chinese_locale': false 中的 false 为 true

5.2、修改 chrome/content/options/browser.js 第 838 行， 修改 'has_chinese_locale': false 中的 false 为 true

6、将修改后的文件重新使用 zip 压缩, 命令：

7、然后将新的 readable@evernote.com.xpi 替换到 ~/.mozilla/firefox/sfhpk2t0.default/extensions 目录下。

8、重新启动 firefox，找篇你要保存的网页，点击 clearly 图标， 然后点击 clip 按钮， 会要求你登录账户， 看见没有，出现了喜闻乐见的 “切换到印象笔记”.

点击 ”切换到 印象笔记“ 之后， 页面可能没有变化， 但是这时候已经可以用你的印象笔记账户登录了。 最后， 如果也使用 64 位 firefox 的， 可以下载我修改好的该插件.

在wr703n下安装配置openwrt、openvpn

Feb 24th, 2013

距离上一篇文章快一年了……，最近由于原来的路由器被小强烧毁，买了一个wr703n路由器，顺便在上面折腾一下openwrt和openvpn，这篇文章记录一下过程,以后关于这个路由器的配置和折腾过程都会更新在这里.

准备工作

给路由器上电之后登陆配置页面看到生产批号是Build 120925,在openwrt wiki页面得知该路由器的硬件版本是1.7而不是路由器背面标注的1.6, 因此还是下载trunk版本的固件免得刷成砖头还需要使用ttl线修复。

下载地址在这里.

由于是从tp原厂固件升级到openwrt，因此下载factory版本，由于jffs2版本刷了之后经过测试占用空间过大无法装其他软件，因此用squashfs版本.

刷机

先用无线连接路由器，下载openwrt-ar71xx-generic-tl-wr703n-v1-squashfs-factory.bin后在路由器的默认管理页面中找到系统工具-升级的选项，然后使用下载的bin文件进行升级，等到页面显示重启后说明刷机结束.
